Options for Bounded Video Playback

Option 1: Animated GIFs (Current - Working)

GIFs are automatically bounded: ?start=3.3&end=6.6

Pros: Simple, auto-loops, no JS needed. Cons: Lower quality, larger file size.

Option 2: Mux Clips (Separate Assets)

Create 3 separate Mux assets from the source video.

Pros: Full video quality, independent streams. Cons: 3x cost, more API calls.

Option 3: JavaScript-Controlled Playback

Use HLS.js to control playback bounds with JS.

Pros: Single asset, full quality. Cons: More complex, requires JS.
